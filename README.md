# 🧪 WESTPA Simulation Dashboard

**A real-time Streamlit dashboard to track WESTPA simulation logs**, with elegant UI, auto-refresh capability, and step-wise progress tracking.  
Made with ❤️ by **Youraj (IIT Patna)**

---

## 🔥 Features

- Real-time log parsing from `simulation.log` file
- Auto-refresh every 5 seconds using `streamlit-autorefresh`
- Displays:
  - ✅ Steps Completed (Progress %)
  - ⚠️ Error Count
  - 📄 Live Log Output
- Minimal, modern, and responsive UI
- Easy to customize and extend
